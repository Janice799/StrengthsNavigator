## 2. 브랜드 포지셔닝

### 포지셔닝 스테이트먼트

```
강점 기반 코칭을 하는 전문 코치들을 위한
StrengthsNavigator Pro는

고객 관리부터 세션 기록, 강점 카드 발송까지
모든 코칭 프로세스를 하나의 플랫폼에서 관리할 수 있는
올인원 코칭 CRM입니다.

일반적인 CRM이나 수기 관리와 달리,
강점 코칭에 최적화된 기능과 워크플로우를 제공하여
코치가 코칭에만 집중할 수 있게 합니다.
```

### 경쟁 우위 (Unique Value Proposition)

#### 우리만의 차별점:
1. **강점 특화** 
   - Gallup 34 강점 기반
   - 강점별 분석 & 인사이트
   
2. **카드 시스템**
   - 감성적 터치
   - 디지털 + 감성 결합
   
3. **올인원 통합**
   - CRM + 세션 + 카드 + 분석
   - 여러 툴 불필요

4. **코치 중심 설계**
   - 코치 언어로 설계
   - 코치 워크플로우 최적화

## 7. 성장 전략 📈

### 바이럴 루프 (Viral Loop)

```
1. 코치 가입
   ↓
2. 고객에게 강점 카드 발송
   ↓
3. 고객이 카드 받음
   ↓
4. 카드 하단: "Powered by StrengthsNavigator"
   ↓
5. 다른 코치들 관심
   ↓
6. 새로운 코치 가입 (반복)

K-Factor 목표: 1.2
(사용자 1명 → 1.2명 유입)
```

### 유지 전략 (Retention)

```
온보딩:
□ 환영 이메일 시퀀스
□ 인터랙티브 튜토리얼
□ 첫 7일 체크인

참여 유도:
□ 주간 인사이트 리포트
□ 월간 성과 요약
□ 새 기능 공지

위기 방지:
□ 사용 안 할 때 알림
□ 해지 전 1:1 상담
□ 피드백 보상
```

### 업셀 전략 (Upsell)

```
Free → Pro:
- 고객 20명 도달 시 팝업
- 강점 분석 리포트 미리보기
- Pro 기능 시범 사용 (7일)

Pro → Team:
- 두 번째 코치 추가 시 제안
- 팀 협업 가치 강조
- 할인 프로모션

Team → Enterprise:
- 10명 초과 시 자동 제안
- 맞춤 기능 소개
- 영업팀 연락
```
